VCFs and panels

How associate a VCF with the panel

  • Place the VCF files in the variant_files directory
  • Navigate to the "Input" tab
  • Select "Add VCF to panel
  • Select the panel you would like the VCF associated with
  • Select the VCF file(s) you would like to associate with the panel

Additional info

VCFs can be associated with multiple panels, each considering the variants only found in their corresponding genes. {% endblock %}
