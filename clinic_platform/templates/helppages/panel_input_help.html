Whats is a panel?

A panel is a selection of genes and their associated data in the form of files (VCF & BAM) and metadata (phenotype) This allows you to only consider certain information in your analysis, avoiding incidental findings.

If you want to extract a subset of genes from the organisms genome

This is useful if you have a list of genes and wish to use them as a panel. This will extract the associated transcripts, and exons.
  • Place the reference gff file in the "ref_genome" folder
  • Select "Gene input", then "Add gene list"
  • Provide a panel id (just a name for the panel) and any phenotype information
  • Add genes to the "Gene list" window, using the gene names as used in the reference gff
  • Select any bam files associated with this panel if applicable (for coverage calculations)
  • Select the annotation gff file from the drop down menu
  • Click submit
